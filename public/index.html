Ejercicio Promedio


Para este ejericio ingrese una pareja de valores enteros(a , b), con los cuales se calculará el promedio.

El error mencionado, aunque no es un error, pues la función funcionaba, en su lugar no era el resultado esperado el que se obtenia, se hallaba en la jerarquia de operadores, pues, el operador de división es el primero en ejecutarse, y este dividia primero a b, y luego sumaba a, por lo cual se obtenia el resultado que no era deseado, se soluciona o se arregla para obtener el resultado del promedio deseado es aplicar paréntesis para la suma, para que esta operación se desarrolle antes, pues el desarrollo de los parentesis es la operación que primero se ejecuta.

Ingrese un valor entero para a .

Ingrese un valor entero para b .

Ejercicio de suma de arreglos


Cree un método que reciba una matriz de solo números (positivos y negativos), con el fin de retornar un objeto con las siguientes propiedades:
1. La suma total de la matriz
2. La suma de sus enteros positivos
3. La suma de los números pares
4. La suma de los números impares Importante,
si la matriz no contiene nada, el resultado debe ser 0.

Ingrese un valor entero para las filas de la matriz.

Ingrese un valor entero para la cantidad de columnas.

Ejericicio tracking Cooridandora


Se requiere crear un método, el cual permita obtener por medio de una respuesta el tracking operativo referente a una guía en específico. Para lograr dicha respuesta, se deberá consumir dos servicios expuestos los cuales arrojaran:

Guias:
https://api.coordinadora.com/cm-model-testing/api/v1/talentos/

Checkpoint: (Tracking)

https://api.coordinadora.com/cm-model-testing/api/v1/talentos/checkpoint

El endpoint que consume el método deberá solicitar e identificar un número de guia (codigo_remision), el cual está compuesto por 11 dígitos o una etiqueta 1d el cual está compuesto por 15 dígitos y siempre comienza con el número 7.

Esta solicitud debe ser a traves de parametro en la url ejemplo: localhost:8080/challenge/tracking/{codigo}

Al finalizar el método deberá resolver el siguiente mensaje de acuerdo al código que se envió.






Ejericicio Score Array


En un método, a partir de una matriz de números enteros enviada por parámetros, retorne una puntuación total basada en los siguientes criterios:

1. Agrega 1 punto por cada número par en el arreglo.
2. Agrega 3 puntos por cada número impar en el arreglo, exceptuando el número 5
3. Agrega 5 puntos a cada número 5 que aparezca en el arreglo.

Nota, el 0 es considerado par.
